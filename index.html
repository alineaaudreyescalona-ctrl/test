Course Description:

The course covers the standard data representation and algorithms to solve computing problems efficiently (with respect to space requirements and time complexity of algorithms). This covers the following: Stacks, Queues, Trees, Graphs, Maps, and Sets. Thorough discussion of sorting and searching algorithms and hashing is covered.

Learning Paths

Foundations of Data Structures and Algorithm Analysis

Understanding how data is represented and how algorithm efficiency is measured.

Linear Data Structures

Organizing and manipulating data in sequential forms.

Non-Linear Data Structures

Managing hierarchical and network-based relationships.

Advanced Algorithms and Optimization Techniques

Developing efficient solutions for large-scale computational problems.
